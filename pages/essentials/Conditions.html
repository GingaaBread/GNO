Conditions

Conditions allow you to execute code depending on the evaluation of a boolean expression

If Condition

The if condition evaluates a boolean expression and only executes the code body if the evaluation is true 

                
                    main class GNOTest {
                        int a = 2

                        if (a is 2) {
                            print("Hello World")
                        }

                        if (a is 5) {
                            print("Are you still there?")
                        }

                        // Output: "Hello World"
                    }

The round brackets within the if-clause ( ) are optional. The parser will try to evaluate the section between the if and the condition bracket { as a boolean expression. 

                
                    main class GNOTest {
                        int a = 2

                        if a is 2 {
                            print("Hello World")
                        }

                        if a is 5 {
                            print("Are you still there?")
                        }

                        // Output: "Hello World"
                    }

Else If Condition

The else if condition specifies a new condition if the former if or else-if clauses evaluated to false. It will not execute its body, if a former if or else-if clause evaluated to true. 

                
                    main class GNOTest {
                        int a = 2
                        int b = 5

                        if (a is 0) {
                            print("Let's see...")
                        }
                        else if (a is 2) {
                            print("Hello World")
                        }
                        else if (b is 5) {
                            print("Are you still there?")
                        }

                        // Output: "Hello World"
                    }

Else Condition

The else condition will execute code if no former if or else-if clause evaluated to true. It will not execute its body, if a former if or else-if clause evaluated to true. 

                
                    main class GNOTest {
                        int a = 2

                        if (a is 0) {
                            print("Let's see...")
                        }
                        else if (a is 5) {
                            print("Hello World")
                        }
                        else {
                            print("Something else...")
                        }

                        // Output: "Something else..."
                    }

Switch

The switch condition works very similar to if and else if statements combined into one construct. 

                
                    main class GNOTest {
                        AnimalType animalType = AnimalType.DOG

                        switch (animalType) {
                            case AnimalType.DOG: 
                                print("Woof!")
                            case AnimalType.WOLF:
                                print("?")
                            default:
                                print("This animal doesn't seem to have a sound!")
                        }
                    }

Just like with if constructs, you can also leave out the parantheses () surrounding the expression:

                
                    main class GNOTest {
                        AnimalType animalType = AnimalType.DOG

                        switch animalType {
                            case AnimalType.DOG: 
                                print("Woof!")
                            case AnimalType.WOLF:
                                print("?")
                            default:
                                print("This animal doesn't seem to have a sound!")
                        }

                        // Output: "Woof!"
                    }
Users familiar with other languages may notice that there are no breaks in this example. This is because, by default, switch cases will execute code until the parser finds a new case or the default case. If you want to use a break in order to, for example, use fallthrough cases, add the break keyword before the switch keyword: 
                
                    main class GNOTest {
                        AnimalType animalType = AnimalType.DOG

                        break switch animalType {
                            case AnimalType.DOG: 
                                print("Woof!")
                            case AnimalType.WOLF:
                                print("?")
                                break
                            default:
                                print("This animal doesn't seem to have a sound!")
                        }
                    
                        // Output:
                        // "Woof!"
                        // "?"
                    }

You can also use or cases to group together similar behaviour:

                
                    main class GNOTest {
                        AnimalType animalType = AnimalType.DOG

                        switch animalType {
                            case AnimalType.DOG or Animal.CAT: 
                                print("I'm a pet!")
                            case AnimalType.WOLF:
                                print("I'm probably not!")
                        }
                    
                        // Output: "I'm a pet!"
                    }

Is Switch

You can combine the is keyword with a switch construction to evaluate a certain value. Note that the identity operator # refers to the default case. 

                
                    main class GNOTest {
                        int res is switch expr {
                            1 => 256,
                            2 => 28,
                            3 => 27,
                            # => -1
                        }
                    }
