Operators

Operators enable you to manipulate data. Here is an overview of operators in GNO:

Comparison Operators

Lesser Than Operator <

The < operator returns true if the left operand is lesser than the right operand. Else, it returns false 

            
                main class Example {
                    print(5 < 3) // false
                    print(3 < 5) // true
                    print(3 < 3) // false
                }

Greater Than Operator >

The > operator returns true if the left operand is greater than the right operand. Else, it returns false 

            
                main class Example {
                    print(5 > 3) // true
                    print(3 > 5) // false
                    print(3 < 3) // false
                }

Divides Operator |

The divides | operator returns true if the left int operand divides the right int operand. Else, it returns false

main class Example { print(3|5) // false print(3|6) // true print(3|9) // true print(2.4|9) // ERROR: 2.4 is not an int }

Lesser Than Or Equal To Operator <=

The <= operator returns true if the left operand is lesser than or equal to the right operand. Else, it returns false

main class Example { print(5 <= 3) // false print(3 <= 5) // true print(3 <= 3) // true }

Greater Than Or Equal To Operator >=

The >= operator returns true if the left operand is greater than or equal to the right operand. Else, it returns false

main class Example { print(5 >= 3) // true print(3 >= 5) // false print(3 >= 3) // true }

Equal To Operator == / is

The ==/is operator returns true if the left operand is equal to the right operand. Else, it returns false

main class Example { print(5 == 3) // false print(5 is 3) // false print(3 == 5) // false print(3 is 5) // false print(3 == 3) // true print(3 is 3) // true string hello = "World" print(hello == "Hello") // false print(hello == "World") // true }
A boolean is already a true or false expression, so instead of checking for equality like 'myBoolean == true' you can use the boolean directly: 'myBoolean'

It is also used to check if a variable is of a certain type. It returns true if the variable is of the specified type.

main class Example { print(5 is int) // true print(5d is int) // false myObject = new MyObject() print(5 is MyObject) // false print(myObject is MyObject) // true myParentObject = new MyParentObject() // Parent of the MyObject class print(myParentObject is MyObject) // true }

Case Equal To Operator ~= / equals

The ~=/equals operator returns true if the left operand is equal to the right operand after casting the type of the right operand to the type of the left operand. Else, it returns false

main class Example { print(5 ~= "3") // false print(5 equals "3") // false print(3 ~= "3") // true print(3 equals "3") // true print(3 ~= 3.5d) // true print(3 equals 3.5d) // true }

Internally, GNO first casts the right operand to the left and then checks for equality. Thus, print(5 ~= "3") evaluates to print(5 == (int) "3")
