Operators

Operators enable you to manipulate data. Here is an overview of operators in GNO:

Logical Operators

Not Operator ! / not

The ! / not operator returns a negated boolean, thus a boolean with the value true returns false, a boolean with the value false returns true. Note that the negated value is only returned, a boolean variable will not be assigned to its negated value (this can be done using the ~ operator discussed in Special Operators).

main class Example { print(!true) // false print(not true) // false print(!false) // true print(not false) // true boolean a = true print(!a) // false print(a) // true }

And Operator && / and

The &&/and operator returns true if both operands are true. If one or both are false, it returns false.

main class Example { boolean learningGNO = true boolean havingFun = true print(learningGNO && havingFun) // true print(learningGNO and havingFun) // true print(learningGNO and !havingFun) // false print(!learningGNO and havingFun) // false }

Or Operator || / or

The ||/or operator returns true if at least one operand is true. If both are false, it returns false.

main class Example { boolean learningGNO = true boolean havingFun = true print(learningGNO || havingFun) // true print(learningGNO or havingFun) // true print(learningGNO or !havingFun) // true print(!learningGNO or !havingFun) // false }

XOR Operator ^ / xor

The ^/xor operator returns true if exactly one operand is true. If both are false, or true, it returns false.

main class Example { boolean learningGNO = true boolean havingFun = true print(learningGNO ^ havingFun) // false print(learningGNO xor !havingFun) // true print(!learningGNO xor havingFun) // true print(!learningGNO xor !havingFun) // false }

Eager And Operator &

The eager & operator behaves like the lazy And Operator &&/and, but still evaluates the right side even if the left side has already been evaluated to false.

Main() { print(false and Test()) // false print(false && Test()) // false print(false & Test()) // false; "Executed: Test()" } boolean Test() { print("Executed: Test()") return true }

Eager Or Operator |

The eager | operator behaves like the lazy Or Operator ||/or, but still evaluates the right side even if the left side has already been evaluated to true.

Main() { print(true or Test()) // true print(false || Test()) // true print(true | Test()) // true; "Executed: Test()" } boolean Test() { print("Executed: Test()") return true }

Logical Operator Order

Consider the following table with operators sorted from highest to lowest precedence.

# Operator
1 Not: ! / not
2 Eager And: &
3 XOR: ^
4 Eager Or: |
5 Lazy And: && / and
4 Lazy Or: || / or
Use parantheses () to change the evaluation order
