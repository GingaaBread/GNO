Operators

Operators enable you to manipulate data. Here is an overview of operators in GNO:

Special Operators

Negation Operator ~

The ~ operator negates a boolean variable without returning it.

main class Example { boolean errorOccured = false ~errorOccured print(errorOccured) // true ~errorOccured print(errorOccured) // false print(~errorOccured) // ERROR: ~errorOccured is not a boolean }

Exists / !Exists Operator

The exists shortcut operator/keyword checks if a variable contains a non-null value. It evaluates to X != null.

main class Example { MyObject myObject print(myObject exists) // false myObject = new MyObject("Hello World!") if (myObject exists) { string greeting = myObject.greeting print("Greeting: " + greeting) } // "Greeting: Hello World!" }

The !exists shortcut operator/keyword checks if a variable contains a null value. It evaluates to X == null.

Ternary Operator ?:

The A ? B : C operator checks a Boolean expression A and then returns the result of one of two expressions depending on whether A was true or false.

main class Example { i = 404 string errorType = i == 404 ? "Not Found" : "Error" print(errorType) // "Not Found" }

Evaluation Operator ? ?: :

The E0 ? E1 ?: E2 ?: E... : B operator checks a range of Boolean expressions E... and then returns the result of the one expression that evaluated to true. If none evaluated to true, it returns the result of the expression in B. This resembles an inline if / else if / else. The evaluate() function can be used for this purpose as well.

main class Example { int errorType = 404 string message message = errorType == 401 ? "Unauthorized" ?: errorType == 402 ? "Payment Required" ?: errorType == 404 ? "Not Found" : "Error" // = message = errorType.evaluate([401:"Unauthorized", 402:"Payment Required", 404:"Not Found", "Error"]) // = if (errorType == 401) { message = "Unauthorized" } else if (errorType == 402) { message = "Payment Required" } else if (errorType == 404) { message = "Not Found" } else { message = "Error" } }

Arrow Operator =>

The => operator is used in lamda expressions and expression bodies.

button.SetOnClick(() => print("Clicked!")) ... string GenerateString() => "Hello World"

Identity Operator #

The # operator has a wide range of meanings in different contexts. For example, in streams, it references the current element, in foreach loops, it references the index, etc.

The functionality of the identity operator is always explained in the respective section.

customers.over.filtered(#.name.StartsWith("To")).all(print) foreach (customer in customers) { string name = customer.name string billingListName = billingList.Get(#) print($"Customer Name: {name}; Billing List Name: {billingListName}") }

Variety Operator :

The : operator has a wide range of meanings in different contexts. For example, you can use :properties to bundle together properties, use it to separate elements in the compare() function, etc.

The functionality of the variety operator is always explained in the respective section.

int errorType = 404 string message = errorType.compare([401:"Unauthorized", 402:"Payment Required", 403:"Not Found", "Error"])
