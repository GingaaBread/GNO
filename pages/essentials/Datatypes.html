Datatypes

Variables are one of the, if not the, most fundamental parts of programming. Generally speaking, a variable is just some kind data. For example, a variable might be age, height,colour, etc...

The value is assigned by using the equality sign = character and then the value. For example, age = 25, height = 12.5, colour = "Red"

Remember: The value comes after the equality sign!

GNO uses the following datatypes:

  • int: 32 bit signed integers
  • long: 64 bit signed integers
  • single: single precision floating point
  • double: double precision floating point
  • char: utf-8 characters
  • string: text
  • boolean: true or false

It is optional to explicitly declare the datatype when directly assigning a value, however, you have to declare the datatype when you declare a variable, but don't assign a value: 

                
                    main class GNOTest {
                        // GNO automatically determines number to be an int, and greeting to be a string
                        
                        number = 25
                        number = 13
                        number = "Is this allowed?" // ERROR -> number was initialised as an int 
                        
                        greeting = "Hello"
                        greeting = "Hi"
                        greeting = false // ERROR -> greeting was initialised as a string 

                        // You can explicitly declare the datatype
                        int age = 99 
                        age = 25
                        age = 5.5 // ERROR -> age was initialised as an int 

                        string colour = "Blue üîµ"

                        // You can also "just" declare a variable and later assign a value
                        boolean happy
                        hapy = true
                        happy = false
                        
                        // However, then you HAVE to declare a datatype!
                        letter // ERROR
                        letter = 'C'
                    }

In the following, you can find an overview of each datatype.

int

int is a 32 bit signed two's complement integer

The value ranges from -231 (minimum) to 231-1 (maximum).

Default Value: 0

                
                    main class GNOTest {
                        int i
                        i = 5
                        i = 2**31-1 // ** = power, thus i=2^31-1 i = 2147483647, MAX int value
                        i = -2**31 // ** = power, thus i=-2^31 i = -2147483648, MIN int value
                    }

long

There may be cases where 32 bit ints don't suffice: a long is a 64 bit signed two's complement integer

The value ranges from -263 (minimum) to 263-1 (maximum).

Default Value: 0l

                
                    main class GNOTest {
                        long i
                        i = 5
                        i = 2**63-1 // ** = power, thus i=2^63-1, MAX long value
                        i = -2**63 // ** = power, thus i=-2^63 i, MIN long value
                    }
You can add an optional L (lowercase or uppercase) after your long value to signalise that you are using a long value: long example = 65261L

single

A single is a 32 bit signed IEEE 754 floating point

Default Value: 0s

                
                    main class GNOTest {
                        single i
                        i = 7.5
                        i = .5 // For 0.X we can leave out the 0
                        i = -285.52
                    }
You can add an optional S (lowercase or uppercase) after your long value to signalise that you are using a long value: single example = 65.25s

double

A double is a 64 bit signed IEEE 754 floating point

Default Value: 0d

                
                    main class GNOTest {
                        double i
                        i = 89247.5
                        i = .5 // For 0.X we can leave out the 0
                        i = -89.5278765
                    }
You can add an optional D (lowercase or uppercase) after your long value to signalise that you are using a long value: single example = 65.25d

char

A char is a single utf-8 character enclosed by single quotes (')

Default Value: ''

                
                    main class GNOTest {
                        char c
                        c = 'X'
                        c = '2'
                        c = '%'
                        c = 'üõ©Ô∏è'
                    }
Some characters need to be escaped! For example, you CANNOT do: char example = ''' because the single quote is needed for the char. Escape Characters:
  • \t: A tab
  • \b: A backspace
  • \n: A newline
  • \r: A carriage return
  • \f: A form feed
  • \': A single quote
  • \": A double quote
  • \\: A backslash

string

A string is a sequence of utf-8 characters enclosed by double quotes (")

Default Value: ""

                
                    main class GNOTest {
                        string t
                        t = "Hello"
                        t = "World"
                        t = "123456789$%&()üßô‚Äç‚ôÇÔ∏è"
                    }

boolean

A boolean is either true or false

Default Value: false

                
                    main class GNOTest {
                        boolean b
                        b = true
                        b = false
                        b = 5 is 2+3 // true
                        b = 2 is 1+1 and 2 is 0+0 // false

                        /* 
                            We can use any boolean expression that results in a clear true or false 
                            Either 5 is the same as 2+3 or not
                            We will take a look at operators next
                        */
                    }
