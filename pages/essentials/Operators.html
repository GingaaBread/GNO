Operators

Operators enable you to manipulate data. Here is an overview of operators in GNO:

Mathematical Operators

Sum Operator +

The + operator is used to add two numbers and returns the sum.

main class Example { print(2+2) // 4 print(2.5d+2) // 4.5d print(5s+3.3s) // 8.3s }

Difference Operator -

The - operator is used to subtract two numbers and returns the difference.

main class Example { print(3-2) // 1 print(2.5d-2.5d) // 0d print(2-10) // -8 }

Product Operator *

The * operator is used to multiply two numbers and returns the product.

main class Example { print(3*2) // 6 print(2.5d*2) // 5d print(20d*10d) // 200d }

Division Operator /

The / operator is used to divide two numbers and returns the division.

main class Example { print(5/2) // 2 print(5d/2d) // 2.5d print(5+2d) // 2.5d }

Power Operator **

The ** operator is used to compute the power of an integer.

main class Example { print(5**2) // 25 print(10**0) // 1 print(3**3) // 27 }

Modulo Operator %

The % operator is used to compute the remainder of an integer after dividing the left operand by the right operand.

main class Example { print(5 % 2) // 1 print(10 % 2) // 0 print(2 % 10) // 2 }

Unary + and -

The unary + returns the vaule of its operand. The unary - returns its numeric negation

main class Example { a = 5 print(+a) // 5 print(-a) // -5 print(-(-a)) // 5 }

Compounds

The operators +, -, *, /, %, ** all have a compound a [operator]= b that evauluates to a = a [operator] b 

                
                    main class Example {
                        a = 1
                        a += 2 // a = 3; equals a = a + 7
                        a -= 2 // a = 1; equals a = a - 3
                        a *= 5 // a = 5; equals a = a * 5
                        a **= 2 // a = 25; equals a = a ** 2
                        a %= 20 // a = 5; equals a = a % 20
                        a /= 2 // a = 2; equals a = a / 2
                    }

Prefix & Postfix Operators

The two operators ++ and -- can be used to increment (++) or decrement (--) a number by 1. The result of the prefix operation is the value before the operation, the result of the postfix operation is the value after the operation.

main class Example { i = 5 // Prefix Operators print(++i) // 6 print(i) // 6 print(--i) // 5 print(i) // 5 // Postfix Operators print(i++) // 5 print(i) // 6 print(i--) // 6 print(i) // 5 }
Note that you cannot use both operators (for instance ++i++) simultaneously!

Cardinality Operator |

For Arrays and other data Structures like Lists, Stacks, and Queues, using the cardinality operators |[Data Structure]| will return the amount of elements in the data structure. This uses the length/size property.

main class Example { int[] a = [1,2,3,4,5] print(|a|) // 5 print(a.length) // 5 print(|[]|) // 0 }
Arrays are discussed in a later chapter
