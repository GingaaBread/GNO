GNO is an object-oriented programming language designed to be convenient. It's three main focuses are:

  • Simplicity
  • Readability
  • Functionality

Despite its object-oriented approach, GNO was created with functional programming in mind.

Hello World!

Create a blank .gno file on your computer like GNOTest.gno

Using a text editor, add the following code to your file:

                
                    main class GNOTest {
                        print("Hello World! üíô")
                    }

If the console displays Hello World! üíô, you are all set!

Optionals

Similar to other languages, semicolons are completely optional in GNO. This way you are not forced to use semicolons, but still can if you want to.

You also don't have to explicitly specify variable types if you don't want to. If you omit them, GNO will determine them automatically! 

                
                    main class GNOTest {
                        food = "Water Melon üçâ"
                        string drink = "Water üåä"

                        print("My favourite food is " + food); // Output: "My favourite food is Water Melon üçâ"
                        print("My favourite drink is " + drink) // Output: "My favourite drink is Water üåä"
                    }

Text Alternatives

GNO offers a multitude of text alternatives to operators, such as and instead of && or is instead of ==. But of course you can still use the traditional operators if you prefer them over the text alternatives! 

                
                    main class GNOTest {
                        i = 0
                        j = 25
                        print(i is 0) // true
                        print(j is 25) // false

                        print(i == 0) // true
                        print(j == 25) // false
                    }

Comfortable Object Orientation

GNO is designed to be an object-oriented programming language without the typical 'writing mania'. It emphasises simplicity and attempts to minimise the amount of 'thoughtless' object-oriented code like constructors, getter & setter methods, etc. 

                
                    class Car : Vehicle, IBuyable {
                        :properties {
                            string colour
                            double width
                            int:get maxVelocity
                        }

                        overriden Drive() {
                            print("Out of the way, yall! üöó")
                        }

                        -Honk() {
                            print("Meep meep!")
                        }

                        boolean Buy() {
                            print("Successfully bought the car!")
                            return true
                        }
                    }
                
            
                
                    main class Example {
                        car = new Car("Red", 2.45, 210)

                        car.Drive() // Output: "Out of the way, yall! üöó"
                        car.colour = "Blue"
                        print(car.colour) // Output: "Blue"
                    }

Functionality

Friends of the functional programming approach will also cherish GNO's focus on comfort and simplicity.

                
                    main class GNOTest {
                        studentAges = [16,19,20,26,36]
                        
                        // Task: Print all grown-up students (age 18+)
                        studentAges.over.filtered(#>17).all(print) // Output: 19,20,26,36

                        // Task: Get the sum of the age of all students in one year 
                        int sum = studentAges.over.mapped(#+1).reduce(sum) // Output: 122

                        // Task: Print all ages except the two youngest
                        studentAges.over.sorted().skip(2).all(print) 
                    }
