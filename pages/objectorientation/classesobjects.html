Classes & Objects

Classes

Classes are created using the class keyword. By default, they are public.

                
                    class Fraction {
                    }

Classes can contain properties, constructors, events, and methods.

                
                    class Fraction {
                        // Properties
                        private int numerator
                        private int denominator

                        // Constructor
                        constructor(int n, int d) {
                            numerator = n
                            denominator = d
                        }

                        // Methods
                        double ToDouble() {
                            return numerator / denominator
                        }

                        overridden String ToString() {
                            return `${numerator} / ${denominator}` 
                        }
                    }

GNO allows using structures to simplify this process:

                
                    class Fraction {
                        :properties {
                            int numerator
                            int denominator
                        }

                        constructor(:properties)

                        :methods {
                            double ToDouble() => numerator / denominator
                            overridden String ToString() => `${numerator} / ${denominator}` 
                        }
                    }

Objects

Objects are specific instances of classes.

                
                    main class GNOTest {
                        f1 = new Fraction(1,5)
                        f2 = new Fraction(2,5)
                        f3 = new Fraction() // Syntax Error: Needs a numerator and denominator as arguments
                        
                        print(f1.ToDouble()) // 0.2d
                        print(f2.ToDouble()) // 0.4d

                        print(f1) // = f1.ToString() = "1/5"
                        print(f2) // = f2.ToString() = "2/5"
                    }

                    class Fraction {
                        :properties {
                            int numerator
                            int denominator
                        }

                        constructor(:properties)

                        :methods {
                            double ToDouble() => numerator / denominator
                            overridden String ToString() => `${numerator} / ${denominator}` 
                        }
                    }

Main Classes

Your project can contain exactly one class preceded by the main keyword. Main classes act like main methods and hence may online contain code that would be allowed in the main method. 

                    
                        main class GNOTest {
                            print("Hello World"); // "Hello World"
                        }

                        // =
    
                        class GNOTest {
                            main {
                                print("Hello World") // "Hello World"
                            }
                        }

For more information about the individual topics like constructors, methods, etc. take a look at the respective chapter.
